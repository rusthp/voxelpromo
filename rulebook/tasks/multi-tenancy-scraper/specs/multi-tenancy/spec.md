# Multi-Tenancy Scraping Specification

## ADDED Requirements

### Requirement: Scraper Execution Isolation
The system MUST isolate execution environments for scraper requests to prevent cross-tenant data leakage.

#### Scenario: Concurrent Affiliate Link Generation
Given two different users requesting affiliate links simultaneously
When the scraper launches the browser instance
Then the system MUST execute both processes in parallel creating separate incognito contexts with their respective cookies

### Requirement: Tenant-Specific API Configuration Loading
The system MUST dynamically load the API credentials and cookie settings from the active user's settings profile before calling external product APIs.

#### Scenario: Automated Collector Background Job
Given a background collection job generated by the Node-Cron scheduler for User A
When the `CollectorService` runs the `collectFromMercadoLivre` routine
Then it MUST fetch User A's API config securely overriding environment variables
