# Guia de Contribuição

Obrigado por considerar contribuir para o VoxelPromo!

## Como Contribuir

1.  **Reportando Bugs**: Abra uma Issue descrevendo o problema, passos para reproduzir e ambiente (OS, Node version).
2.  **Sugerindo Funcionalidades**: Abra uma Issue com a tag `enhancement` detalhando sua ideia.
3.  **Pull Requests**:
    *   Faça um Fork do projeto.
    *   Crie uma branch para sua feature (`git checkout -b feature/AmazingFeature`).
    *   Certifique-se de que os testes passam (`npm test`).
    *   Siga o estilo de código do projeto (ESLint/Prettier).
    *   Abra o PR descrevendo suas alterações.

## Padrões de Código

*   Usamos **TypeScript** estrito.
*   Commits devem seguir o padrão [Conventional Commits](https://www.conventionalcommits.org/) (ex: `feat: add new provider`, `fix: resolve crash`).
